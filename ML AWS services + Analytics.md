**SageMaker** - is a fully managed service to build, train, and deploy ml models
**Lex** - Conversion interface service. With it you can build voice and text chatbots
**Polly** - text to speech
**Rekognition** - image and video recognition service
**Transcribe** - speech to text
**Textract** - OCR
**Translate** - neural machine learning translation service
**Comprehend** - NLP service. Find relationships between text to produce insights. 
**Kendra** - enterprise ML search engine service. Uses NL to suggest answers to question instead of simple keyword matching


BIG Data analytics
**Amazon Athena** - serverless interactive query service, when you want to query csv and json
**Elastic MapReduce EMR** - data processing and analysis. Can be used for creating reports just like Redshift, more suited to transform unstructured to structured data on the fly.
**Kinesis Data Analytics** - run queries in the real time stream
**MSK** - [[Application Integration]] 
**Redshift** - [[Storage]]
**QuickSight** - Business intelligence dashboards [[Governance + Business]]
**Glue** - Extract, Transform, and Load service. Move data from one location to another. Similar to DMS [[Storage]], but more robust due to the transformations
**AWS Data exchange** - a catalog of 3rd party datasets.
