What tf is provisioning?

The allocation or creation of resources and services to a customer. Responsible for setting up and managing AWS services.

Beanstalk

AWS OpsWorks - configuration management service that also provides instances of the open source configuration managed software Chef and Puppet

AWS Amplify - is a mobile and web-app framework, that will provision multiple AWS services as your backend

AWS App Runner - fully managed services (PaaS for containers) make it easy for devs to quick deploy containerized web apps and APIs at scale with no prior infra

AWS CodeStar - Unified user interface, enabling one to easily manage your software dev activities in one place. 

WHAT tf is BYOL (bring your own license)???

The process of reusing an existing software license to run vendor software on a cloud vendor software on a cloud vendor's computing service. 

---> **AWS License Manager** is a service that makes it easier for you to manage your software licenses from vendors.

CloudWatch - does logs, metrics and events based conditions on a server, alarms, Dashboard
- default logs for 90 days but for anything more you have to set up a TRAIL. Trails output to S3 and don't have a gui like Cloudtrail so use amazon Athena
- Alarms - monitors a CloudWatch Metric based on a defined threshold.
