5 pillars
- Operational Excellence
	- Run and monitor systems
- Security
	- protect data and systems
- Reliability
	- mitigate and recover from disruptions
- Performance Efficiency
	- use computing resources effectively
- Cost Optimization
	- get the lowest price

Component - code, config, and AWS resource against a requirement
Workload - a set o components that work together to deliver business value
Milestones - key changes of you architecture through product life cycle
Architecture - How components work together in a workload
Tech portfolio - a collection of workloads required for the business to operate.
